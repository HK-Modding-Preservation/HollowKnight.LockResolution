# 锁定分辨率(进入存档后 起效)  防止游戏自动改我分辨率

锁定 默认开启

默认分辨率 第一次运行分辨率

失去焦点时垂直同步 默认关闭

比如 VSync1/4 如果屏幕刷新率60 切到别的程序 游戏就会只有15帧
